# LiveCode

LiveCode is a real-time AI-powered collaborative coding platform designed for coding collaboration, online learning, and technical interviews.

The platform allows multiple users to:
- write and edit code together in real time
- communicate through chat/video
- run code and view output instantly
- receive AI-assisted debugging and coding support

## Tech Stack
- React.js
- Node.js
- Socket.IO
- Monaco Editor
- MongoDB
- WebRTC
- Python AI Modules

## Features
- Real-time collaboration
- Multi-language code execution
- AI code assistance
- Learning & interview modes
- Live output preview
- Chat and communication system
